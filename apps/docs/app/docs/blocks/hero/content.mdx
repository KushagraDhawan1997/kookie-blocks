A flexible container for creating impactful hero sections. Use it to create attention-grabbing headers for landing pages, product launches, and marketing content with consistent spacing and typography.

Built on Kookie UI's Flex and Heading components for maximum composability.

## Installation

```bash
npm install @kushagradhawan/kookie-blocks
```

## Usage

```tsx
import { Hero } from '@kushagradhawan/kookie-blocks';

export function MyHero() {
  return (
    <Hero.Root>
      <Hero.Title>Welcome to My App</Hero.Title>
    </Hero.Root>
  );
}
```

## Anatomy

Hero is a compound component with the following parts:

```tsx
<Hero.Root>
  <Hero.Title>{/* Heading text */}</Hero.Title>
  {/* Additional content */}
</Hero.Root>
```

## Root Props

The Root component is a specialized Flex container with vertical layout and spacing optimized for hero sections. Accepts all Flex props from Kookie UI.

| Prop | Type | Description |
| ---- | ---- | ----------- |
| `direction` | `'row' \| 'column' \| 'row-reverse' \| 'column-reverse'` | Flex direction. Defaults to `'column'` |
| `gap` | `'1' \| '2' \| '3' \| '4' \| '5' \| '6' \| '7' \| '8' \| '9'` | Spacing between children. Defaults to `'6'` |
| `p` | `'1' \| '2' \| '3' \| '4' \| '5' \| '6' \| '7' \| '8' \| '9'` | Padding. Defaults to `'6'` |
| `align` | `'start' \| 'center' \| 'end' \| 'baseline' \| 'stretch'` | Align items along the cross axis |
| `justify` | `'start' \| 'center' \| 'end' \| 'between'` | Justify content along the main axis |
| `asChild` | `boolean` | Merge props with child element using Radix Slot pattern |

All other Flex props from Kookie UI are also supported (width, height, display, position, etc.).

## Title Props

The Title component is a Heading optimized for hero sections. Accepts all Heading props from Kookie UI.

| Prop | Type | Description |
| ---- | ---- | ----------- |
| `size` | `'1' \| '2' \| '3' \| '4' \| '5' \| '6' \| '7' \| '8' \| '9'` | Heading size scale |
| `weight` | `'light' \| 'regular' \| 'medium' \| 'bold'` | Font weight |
| `align` | `'left' \| 'center' \| 'right'` | Text alignment |
| `as` | `'h1' \| 'h2' \| 'h3' \| 'h4' \| 'h5' \| 'h6'` | HTML heading element. Defaults to `'h1'` |
| `asChild` | `boolean` | Merge props with child element using Radix Slot pattern |

All other Heading props from Kookie UI are also supported (color, highContrast, trim, etc.).

## Examples

### Centered Hero

Center the hero content for maximum visual impact:

```tsx showLineNumbers
import { Hero } from '@kushagradhawan/kookie-blocks';
import { Text } from '@kushagradhawan/kookie-ui';

export function CenteredHero() {
  return (
    <Hero.Root align="center">
      <Hero.Title>Build Something Amazing</Hero.Title>
      <Text size="5">The fastest way to ship your next project</Text>
    </Hero.Root>
  );
}
```

### With Custom Spacing

Adjust spacing and padding for different layouts:

```tsx showLineNumbers
import { Hero } from '@kushagradhawan/kookie-blocks';
import { Text, Button } from '@kushagradhawan/kookie-ui';

export function SpacedHero() {
  return (
    <Hero.Root p="9" gap="8">
      <Hero.Title size="9">Transform Your Workflow</Hero.Title>
      <Text size="5">Tools designed for modern teams</Text>
      <Button size="4">Get Started</Button>
    </Hero.Root>
  );
}
```

### Full-Width Hero

Create edge-to-edge hero sections:

```tsx showLineNumbers
import { Hero } from '@kushagradhawan/kookie-blocks';
import { Text, Container } from '@kushagradhawan/kookie-ui';

export function FullWidthHero() {
  return (
    <Hero.Root p="0" width="100%">
      <Container size="4" p="9">
        <Hero.Title>Welcome</Hero.Title>
        <Text>Your journey starts here</Text>
      </Container>
    </Hero.Root>
  );
}
```

### With Background

Combine with Kookie UI's material system for depth:

```tsx showLineNumbers
import { Hero } from '@kushagradhawan/kookie-blocks';
import { Box, Text } from '@kushagradhawan/kookie-ui';

export function MaterialHero() {
  return (
    <Box material="translucent" style={{ background: 'var(--accent-3)' }}>
      <Hero.Root>
        <Hero.Title color="accent">Premium Features</Hero.Title>
        <Text highContrast>Everything you need to succeed</Text>
      </Hero.Root>
    </Box>
  );
}
```

### Variant Aliases

Hero.Root has multiple aliases for convenience:

```tsx
import { Hero } from '@kushagradhawan/kookie-blocks';

// All of these are equivalent
<Hero.Root>...</Hero.Root>
<Hero.Variant1>...</Hero.Variant1>
<Hero.One>...</Hero.One>
```

## Accessibility

Hero provides semantic HTML structure for screen readers and search engines:

### Semantic HTML

- Uses `<h1>` by default for Hero.Title (can be customized with `as` prop)
- Proper heading hierarchy when combined with other content
- Semantic Flex container structure

### Keyboard Navigation

- No interactive elements by default
- When containing buttons or links, standard keyboard navigation applies

### Screen Readers

- Hero.Title announces as heading with appropriate level
- Flex layout preserves logical reading order
