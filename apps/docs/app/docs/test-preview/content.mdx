## Multiple Components

Use `background="dots"` to show component boundaries clearly when demonstrating layouts with multiple elements.

<PreviewBlock background="dots">
  <Flex gap="3" align="center">
    <Button variant="solid" size="2">Primary</Button>
    <Button variant="outline" size="2">Secondary</Button>
    <Button variant="ghost" size="2">Ghost</Button>
  </Flex>
</PreviewBlock>

```tsx showLineNumbers
<Flex gap="3" align="center">
  <Button variant="solid" size="2">Primary</Button>
  <Button variant="outline" size="2">Secondary</Button>
  <Button variant="ghost" size="2">Ghost</Button>
</Flex>
```

## Usage

1. Import components and PreviewBlock at the top
2. Wrap rendered component in `<PreviewBlock>`
3. Show code separately in markdown fence
4. Use `background="dots"` for multi-component layouts
5. Use `background="none"` for components with their own backgrounds
